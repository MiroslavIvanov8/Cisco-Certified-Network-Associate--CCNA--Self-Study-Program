# Network Engineering Portfolio

## About This Repository

This repository documents my networking journey from learning the fundamentals of routing and switching to earning the Cisco Certified Network Associate (CCNA) certification in June 2026.

Over the past several years, I have been developing my technical skills through hands-on labs, troubleshooting exercises, personal projects, and structured learning. The goal of this portfolio is to showcase not only the final projects but also the practical experience gained throughout the process.

The repository contains personal networking projects, enterprise design documents, security-focused labs, Packet Tracer topologies, and a complete collection of the networking experiences I completed while preparing for the CCNA exam.

---

## CCNA Learning Path

A significant part of this repository follows the curriculum of the NetSpert Cisco Certified Network Associate (CCNA) Self-Study Program, which provides a structured learning path covering networking fundamentals, routing, switching, security, network services, wireless networking, and automation. The program includes 27 learning sections, extensive hands-on exercises, and practical Cisco Packet Tracer labs designed to build real-world networking skills.

The completed learning path covers topics including:

* Cisco CLI Fundamentals
* Router and Switch Configuration
* IPv4 and IPv6 Networking
* Static Routing
* VLANs and Trunking
* Inter-VLAN Routing
* STP and EtherChannel
* OSPF
* DHCP
* NAT and PAT
* NTP
* Secure Device Administration
* ACLs
* Port Security
* Network Automation Fundamentals
* Enterprise Wireless Networking
